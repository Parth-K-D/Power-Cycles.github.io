Power Cycles

Power cycle is used to power the turbopumps to pressurize the propellants.

Pressure fed "cycle"

This is not really a cycle as it does not power the turbomachinary but serves as a bsis of comparision.
It is the the simplest system,but it is also the most performance-limited if compared to pump-fed systems.

  • It provides us an excellent reliability
  • It also gives us a strong start/shutdown(when hypergolic propellants are used)
  • It has a good storsbility(while using hypergolic propellants)
  • It gives us a good throttleabiltity
  • The performance is acceptable
  • Examples
    • Aerojet AJ10-190
    • Aerojet AJ10-118
    • Most of the RCS/ACS systems

Gas Generator cycle

  • This is one of the first power cycle developed for rocket propulsion
  • This uses either specific or common propellants in gas generator to produce turbine drive gas
  • It has a good reliabilty and robust start/shutdown
  • It can use almost any viable bipropellant combination
  • GG cycle variations
    • Monopropellant GG
    • Bipropellant GG
      1. Series turbine
      2. Parallel turbine

Monopropellant GG

  • It is the early/original power cycle
  • It has an acceptablr performane
  • Independent monopropellant control provides a more reliable system,but this increases the wieght of third propellant
  • Examples
    • A-4(v-2)
    • A-6(Navaho-1)
    • A-7(Redstone)

Bipropellant GG(single TPA)

  • It has a better performance than the monopropellant GG
  • It has been improved by removal of 3rd propellant
  • It works well for propellants with similar fluid properties to allow common shaft RPM
  • Examples
    • F-1
    • Atlas MA-2
    • MC-1/Fastrac

Bipropellant(Series turbine)

  • It allows independent thrust and MR control(Moving Range)
  • It works good for propellants with different fluid properties that require different pump speeds
  • Examples
    • J-2


Bipropellant GG(Parallel turbine)

  • This system allows independent thrust and MR control
  • It works good for propellants with different fluid properties that require different pump speeds
  • Examples
    • Vulcain
    • rS-68



Tapoff cycle

  • In this system,turbopumps are driven by hot gas which are tapped from the main chamber
  • It has a good throttleabiltity
  • This system has a low operational experiance
  • It's simplicity offers potential high reliability
  • Examples
    • J-2S


Expander cycle(Single TPA)

  • The thrust is limited by ability to utilize heated fuel
  • It has a good throttleabiltity
  • There are benign failure modes
  • It gives us a high reliability
  • Examples
    • RL10 family


Expander cycle(Dual TPA)

  • The thrust is limited by ability to utilize heated fuel
  • It has a good throttleabiltity
  • There are benign failure modes
  • It gives us a high reliability
  • In this system,turbine bypasses permit independent thrust and MR control
  • Examples
    • RL60
    • MB-xx

Staged combustion(SC) cycle

  • It utilizes all propellants to generate thurst
  • It has a high performance,but high Isp requires high operating pressure
  • This system has a good reliability,but high operating conditions demand vigilance
  • The high operating pressures in this system usually requires the use of boost pumps to increase propellant pressure entering main pumps.
  • SC cycle variations
    1. Fuel rich(FRSC)
    2. Oxidizer rich(ORSC)
    3. Full flow(FFSC)


Fuel rich SC cycle(Dual preburners)

  • This system permits independent MR and thrust level throttleabiltity
  • Examples
    • SSME(with boost pumps)


Fuel rich SC cycle(Single preburners)

  • It permits some MR and thrust level throttleabiltity
  • This has a better system simplicity and offers better reliability
  • Examples
    • RD-0120
    • LE-7
    • RS-30 ASE(DDT&E incomplete)
    • COBRA(DDT&E incomplete)


Oxidizer rich SC cycle

  • It needs use of materials resistant to ignition in an oxidizer-rich environment(more coating needed)
  • This has a good reliability
  • This system is mostly used in Russia
  • Examples
    • RD-253
    • RD-170
    • RS-84


Full flow SC cycle

  • This system is complex which degrades it's reliability and it increases it's cost
  • It's complicated flow management requires complex transient and mainstage control
  • Examples
    • IPD(Integrated Powerhead Demonstrater)


In addition to these primary cycles which were discussed there are numerous variations which are used to pursue a higher performance
